# Navigation Structure - Account Pages

## Marketplace Header Dropdown Menu

When users click the profile icon in the marketplace header, they now see:

```
┌─────────────────────────────┐
│  My Profile                 │
├─────────────────────────────┤
│  ACCOUNT                    │
│  • Addresses                │
│  • Payment Methods          │
│  • Wishlist                 │
├─────────────────────────────┤
│  MARKETPLACE                │
│  • My Orders                │
│  • My Listings              │
│  • Settings                 │
└─────────────────────────────┘
```

## Account Pages Routes

### Account Section
- `/account/addresses` - Manage shipping & billing addresses
- `/account/payment-methods` - Manage cards & crypto wallets
- `/account/wishlist` - Multiple wishlists with price tracking
- `/account/orders` - Order history & tracking

### Marketplace Section
- `/marketplace/profile` - User profile
- `/marketplace/listings` - Seller listings
- `/marketplace/settings` - Account settings

## How to Access

1. **Click profile icon** in marketplace header (top right)
2. **Hover over the dropdown** to see all options
3. **Click any link** to navigate to that page

## Features

### Organized Sections
- **Account** - Personal data management
- **Marketplace** - Marketplace-specific features

### Visual Separators
- Section headers in uppercase
- Divider lines between sections
- Clean, organized layout

### Consistent Navigation
- All links use the same navigation pattern
- Forced page refresh for consistency
- Works across the entire marketplace

## Mobile Support

The same navigation structure works on mobile devices through the mobile navigation menu.

## Next Steps

Users can now easily access all account pages from anywhere in the marketplace!
